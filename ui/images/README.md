# promt0r Images

Add your image files to this directory:

## Required Files

- **favicon.png** — Icon displayed in browser tab (dark icon version, 256x256 or larger)
- **logo-light.png** — Light version of logo with text (displayed in topbar when on light theme)
- **logo-dark.png** — Dark version of logo with text (for future dark theme support)

## Optional Files

- **logo-icon.png** — Just the icon without text (for other UI elements)
- **banner.png** — Full banner/hero image for documentation

## Placement

This directory structure is already set up in the HTML:
- `index.html` references: `images/favicon.png` and `images/logo-light.png`
- Update paths in `index.html` if you use different filenames
